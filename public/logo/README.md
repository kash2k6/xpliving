# Logo Assets

Place your logo files in this directory.

Recommended formats:
- `logo.png` or `logo.svg` for the main logo
- `logo-dark.png` or `logo-dark.svg` for dark mode variant (optional)
- `logo-light.png` or `logo-light.svg` for light mode variant (optional)

The logo should be optimized for web use and work well at various sizes.
